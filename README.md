# UE_SuperResolution

파이썬 머신러닝 프로젝트\n
언리얼 엔진5를 이용한 이미지 Super Resolution

# Files

Model.ipynb : 메인 코드\n
LR/ : 저해상도 이미지 폴더\n
HR/ : 고해상도 이미지 폴더\n
model/ : Model pth 저장소\n
config.json : train.py args 저장\n
data_loaders.pth : 데이터셋 경로,정보 저장\n
dataset_module.py : 데이터셋 정의\n
model_class_model_cnn6.py : 모델 정의\n
train.py : train 코드
